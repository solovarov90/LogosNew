# 📋 Гайд по оформлению Базы Знаний

## 1. Структура файла (Стандарт)
Каждый файл начинается с YAML-блока. Теги пишем просто и прямо: `#маркетинг`, `#синтез` и т.д.

```markdown
---
type: концепт | новости | синтез | материалы
status: черновик | исследование | база
source: [[...]]
tags: 
  - "#маркетинг"
  - "#исследование"
---
```

## 2. Группы тегов (Для Графа)

### По смыслу:
- `#копирайтинг`
- `#ии_инструменты`
- `#маркетинг`
- `#психология`
- `#кейсы`

### По стадиям:
- `#исследование` (сырые материалы)
- `#кристаллизация` (чистые концепты)
- `#синтез` (готовые болванки для ИИ)

## 3. Настройка групп в Графе (Скопируй в Obsidian)
1. **Зеленый:** `tag:#синтез`
2. **Синий:** `tag:#кристаллизация`
3. **Желтый:** `tag:#исследование`
4. **Красный:** `tag:#психология`
