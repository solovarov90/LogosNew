# ГАЙД: Форматирование для Telegram Бота (Smartposting) 🤖

Этот файл содержит стандарты и примеры сообщений для Telegram-бота, используемые при генерации контента.

## 🛠 Технический стандарт
- **Формат:** HTML
- **Переменные:**
  - `{name}` — Имя пользователя
  - `{username}` — @юзернейм
  - `{date}` — Текущая дата
  - `{time}` — Текущее время
- **Теги:** `<b>`, `<i>`, `<u>`, `<s>`, `<code>`, `<a href="URL">`.
