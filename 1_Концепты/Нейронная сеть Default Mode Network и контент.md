# Нейронная сеть Default Mode Network (DMN) и контент 🧠
Тип: Концепт
Тэги: #нейробиология #DMN #блуждание_ума #сторителлинг

Суть: Default Mode Network активируется, когда мозг не занят конкретной задачей, а "блуждает". Это состояние идеально для инсайтов и глубокого вовлечения. Контент, активирующий DMN, должен быть недирективным, содержать паузы и приглашать к размышлению, а не просто давать инструкции.
