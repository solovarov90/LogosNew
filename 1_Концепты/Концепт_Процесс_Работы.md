# Процесс выстраивания работы с ИИ

Моя техника работы с нейросетями для маркетинга строится на этапах:

1. **Итеративность:** Не просить ИИ написать весь текст сразу. Сначала запрашиваем структуру (скелет).
2. **Бриф:** Явно описываем целевую аудиторию, их боли и страхи. Промпт: "Напиши для аудитории [описание], у которых болит [проблема]".
3. **Ограничения:** Задавать жесткие правила. Промпт: "Не используй слова X, Y, Z. Ограничение в 1500 символов".
4. **Редактура:** ИИ собирает факты, человек добавляет душу.
