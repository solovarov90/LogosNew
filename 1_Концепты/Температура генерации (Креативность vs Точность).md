# Температура генерации (Креативность vs Точность)

## Суть концепта
Параметр "Temperature" определяет степень случайности выбора следующего слова. Низкая температура (0.1–0.3) делает ИИ сухим и точным (для кода/фактов). Высокая (0.7–1.0) — делает его креативным, но склонным к галлюцинациям.

## ИИ-интеграция
Smartposting автоматически меняет температуру на разных этапах петли. Для [[Аналитика намерений аудитории]] используется низкая температура, а для [[Сторителлинг с ИИ]] — высокая. Это позволяет получать "живые" тексты без потери логики.
