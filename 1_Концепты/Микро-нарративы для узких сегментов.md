# Микро-нарративы для узких сегментов

## Суть концепта
Маркетинг для "всех" — это маркетинг ни для кого. Аудитория сегментируется на узкие "племена" по интересам, сленгу и ценностям, к которым нужен свой подход и свой язык.

## ИИ-интеграция
Вместо одного "универсального" поста для всех соцсетей, используется ИИ-размножение контента — [[Генеративная локализация]]. Базовый текст об акции алгоритмически переписывается в 15 версий: для геймеров (с отсылками к играм), для мамочек (через призму безопасности), для студентов (через призму экономии). Это [[Адаптивный Tone of Voice]] возведенный в абсолют.
