# ИИ-аудит продуктовой линейки

## Суть концепта
Использование ИИ для поиска "дыр" в твоей воронке продуктов. ИИ анализирует, где клиент теряет интерес или где не хватает промежуточного продукта для перехода к следующему этапу.

## ИИ-интеграция
ИИ сопоставляет [[Семантический анализ болей]] аудитории с твоим текущим списком офферов. Если есть боли, которые ты не закрываешь — ИИ подсвечивает это как зону для запуска нового продукта.
