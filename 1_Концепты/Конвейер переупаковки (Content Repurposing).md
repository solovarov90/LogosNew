# Конвейер переупаковки (Content Repurposing)

## Суть концепта
Создавать хороший контент дорого. Глупо использовать часовое интервью или глубокую статью только один раз. Контент должен работать во всех форматах и на всех платформах.

## ИИ-интеграция
ИИ решает проблему стоимости дистрибуции через [[Масштабирование смыслов]]. Загрузив одно 40-минутное YouTube-видео в ИИ-пайплайн (Whisper + Claude + Midjourney), команда за час получает: транскрипцию, лонгрид для блога, 5 коротких постов для Telegram с разными [[Визуальные якоря]], серию писем и выжимку главных инсайтов. Один смысловой актив алгоритмически дробится на [[Омниканальная сетка контента]].
