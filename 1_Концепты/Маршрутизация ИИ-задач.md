# Маршрутизация ИИ-задач

## Суть концепта
Если у тебя есть несколько ИИ-агентов (один для поиска, другой для стиля, третий для проверки фактов), их нужно направлять (маршрутизировать) на конкретные этапы [[Итеративный Бриф]].

## ИИ-интеграция
Smartposting выполняет роль Центрального Диспетчера. Он знает, что после [Агента поиска] нужно обязательно запустить [Агента-редактора], а затем [Агента-финалайзера]. Это и есть твоя нелинейная система.
