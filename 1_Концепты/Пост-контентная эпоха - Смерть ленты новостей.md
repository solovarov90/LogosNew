# Пост-контентная эпоха: Смерть ленты новостей

## Суть концепта
Пользователи устанут от бесконечного скроллинга. Они будут потреблять информацию только тогда, когда она решает их *конкретную, сиюминутную* задачу. Лента как основной канал уйдет.

## ИИ-интеграция
Контент должен доставляться через [[ИИ-ассистент как зеркало идей]] по запросу, а не через проталкивание в ленту. Твой контент в базе — это "набор знаний", который ИИ будет вытягивать по мере необходимости клиента.
