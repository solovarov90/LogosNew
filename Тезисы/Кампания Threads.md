# ТЗ для таргетолога — рекламная кампания в Threads

> **Задача:** запустить отдельную кампанию **только на плейсмент Threads** (Meta Ads), которая ловит «нашу» аудиторию — людей, которые уже листают Threads и потенциально видели аккаунт «Техномаг» (solovarovp). Цель — провести их на лендинг `case.smart-posting.ru`.
>
> **Почему именно Threads:** весь результат кейса получен в Threads, продукт и скрины-пруфы — из Threads. Аудитория там «нашего» формата: вдумчивая, интеллектуальная, реагирует на смысл, а не на танцы. Креативы должны выглядеть **нативно** — как обычный пост в ленте Threads (текст от первого лица + скрин), а не как вылизанный баннер.

## 1. Технические настройки кампании

| Параметр | Значение |
|---|---|
| Платформа | Meta Ads Manager |
| **Плейсмент** | **Только Threads** (ручной выбор плейсментов → снять все галки, оставить Threads). Если Threads недоступен соло в кабинете — оставить Threads + Feed Instagram и резать остальное на уровне ставок. |
| Цель кампании | **Трафик на сайт** → лендинг `case.smart-posting.ru` (решено: ведём на лендинг, не сразу в бот) |
| Гео | Россия + СНГ (RU, KZ, BY, по желанию UA-аудитория РФ-языка). Язык аудитории — русский. |
| Возраст | 25–45 |
| Пол | Все |
| Оптимизация | Сначала клики по ссылке; после набора данных пикселем — по конверсии на лендинге (переход в бот/клик по CTA лендинга как цель пикселя) |
| UTM | `?utm_source=threads&utm_medium=cpc&utm_campaign=threads_case&utm_content={{креатив}}` — обязательно у каждого крео свой `utm_content`, чтобы видеть, что зашло |

**Структура для теста:** 1 кампания → 5–6 групп объявлений (по числу крео-концептов ниже) с одинаковым бюджетом → A/B тест 4–7 дней → масштабируем победителя, остальное режем.

## 2. Аудитории (предложить таргетологу на выбор / по очереди)

1. **Интересы (холодная):** психология, коучинг, саморазвитие, эзотерика, астрология, нумерология, осознанность, нейросети/AI, ChatGPT, личный бренд, SMM, инфобизнес, Obsidian, продуктивность.
2. **Поведенческая:** активные авторы / создатели контента, мелкие предприниматели, онлайн-эксперты.
3. **Lookalike:** похожие на посетителей лендинга + на пользователей Telegram-бота (если есть пиксель/база — 1–3%).
4. **Ретаргет:** кто был на лендинге, но не дошёл до бота.

> **Важно про модерацию (Meta):** нельзя «угадывать личные качества» в тексте напрямую («ты выгораешь?», «ты инфоцыган»). Формулируем от себя: «я выгорал», «я устал от...». См. раздел 6.

## 3. Банк тезисов (заголовки/хуки — таргетолог комбинирует)

Это «крючки» из брифа заказчика. Их можно ставить как заголовок объявления или первую строку текста:

- **«Самый быстрорастущий экспертный аккаунт в Threads на территории СНГ. Рассказываю, как я это сделал.»** ⚠️ (сильный, но «самый» — рискованный для модерации суперлатив; запасной мягкий вариант ниже)
- «Возможно, вы видели этот аккаунт в Threads. Рассказываю, как поднять такой же — на автопилоте.»
- «2 300 000 просмотров за 90 дней. 0 рублей в рекламу. Вот как.»
- «Как вырастить экспертный аккаунт в Threads, не превращаясь в раба контент-плана.»

## 4. КРЕАТИВЫ — карусели (основной формат)

> **Формат всех крео — карусель** (свайп-галерея). Это даёт «галерею скринов сразу несколько» в одном объявлении, само ведёт по логике кейса и нативно для Threads.
>
> **Тех. требования к карусели:** карточки квадрат **1080×1080** (либо 1080×1350, если кабинет позволяет вертикальную карусель), 2–10 карточек, у каждой — свой файл. **Весь текст-плашка впекается прямо в картинку дизайнером** (в Threads подписи под карточками почти не видны). «Текст объявления» ниже — это общий пост над каруселью. Стиль карточек: тёмная тема, крупная рубленая типографика, лёгкое свечение (как лендинг). Все пути — от корня папки **`Smartposting`** (в Downloads).
>
> Первая карточка = **обложка-хук** (фото + цифра/тезис, должна останавливать скролл). Последняя карточка = **CTA**. Между ними — пруфы.

## 6. Чек-лист безопасности для модерации Meta

Threads-реклама проходит ту же модерацию, что и Facebook/Instagram. Чтобы не словить отклонение:

- ❌ **Нет конкретным суммам дохода** («заработал 300к», «чеки 300к+», «12 человек купили за 10–20к»). Заменяем на «монетизировал», «люди оформляли тарифы».
- ❌ **Нет обещаниям результата** («ты заработаешь», «гарантирую охваты»). Только личный опыт: «у меня получилось», «мой результат».
- ❌ **Не угадываем личные качества** аудитории во 2-м лице («ты выгорел», «ты инфоцыган»). Переводим на себя: «я выгорал».
- ⚠️ **«Самый быстрорастущий»** — суперлатив, могут попросить пруф. Держим наготове мягкую версию «один из самых быстрорастущих» (см. раздел 3).
- ✅ На лендинге уже есть дисклеймер про индивидуальность результатов — в объявлениях тоже избегаем кликбейта про деньги.
- ✅ Ведём на `landing_draft_edited.md`-версию лендинга (Facebook-safe), не на оригинал с суммами.
